# 橋渡しセクション「満月理論から美点発見へ」デザインイメージ

**作成日：** 2026年1月11日  
**目的：** 満月理論 → 美点発見 → アプリ の自然な流れを作る

---

## 📍 挿入位置

### 現在の構成

```
3. ソリューションセクション（満月理論）
   └─ 満月理論の確かな実績

4. シーンストーリーセクション（削除または移動）

5. 3ステップセクション（使い方は簡単）
   └─ アプリの使い方
```

### 問題点

```
❌ 満月理論 → いきなりアプリ という唐突な展開
❌ 「なぜ美点発見なのか」が不明
❌ 「なぜアプリなのか」の必然性がない
```

---

### 修正後の構成

```
3. ソリューションセクション（満月理論）
   └─ 満月理論の確かな実績

【NEW】3.5 橋渡しセクション「満月理論から美点発見へ」
   ├─ Part 1: YSメソッドから生まれた「美点発見」
   ├─ Part 2: 基本形はシートに書き出す方法
   └─ Part 3: 持ち歩けるアプリ形式を開発

4. シーンストーリーセクション（移動または統合）

5. 3ステップセクション（使い方は簡単）
   └─ アプリの使い方
```

```
氏名: 谷 光宏（たに みつひろ）

所属:
全日本空輸株式会社 人事部　ANA人財大学

経歴:
- 関西空港支店総務課長
- 広報室部長
- 大阪空港支店総務部長
- ANAエアーサービス松山 社長
- ANAエアポートサービス 常務取締役

美点発見実践:
当初、美点発見を松山空港にて200名の社員を対象に実施。
その結果、従業員満足度の伸び率ダントツ日本一を達成。
その後、ANAの全社員教育に美点発見を導入。

実践方法:
当時、ANAエアーサービス松山社長として、社員全員の美点発見を始めた時は、
各々社員の美点が思いつかず、ノートが埋まらない状態でした。
ですので、まずは社員一人一人とのコミュニケーションを取っていきました。

だんだんノートが埋まっていくにつれて、どの社員も「会社を愛していること」や
「職場の仲間を大切にしていること」がわかってきて、
社員に対する感謝の気持ちが自然と湧き上がってきました。

成果:
美点発見というのはとてもシンプルな手法ですが、実際にやってみると、
年度末の業績がまるで別会社のように上がりはじめ、
社内調査で社員の満足度が急上昇するなど、
そのような成果はこれまで経験したことがないくらい驚くべきものでした。

その他の成果:
・松山空港では、ANAグループ会社内でそれまで一度も受けたことのなかった賞を、
  美点発見後にいくつも受賞するようになった。
・メンタル不調の部下に上司が美点発見を行ってから面談をすると、
  メンタル不調がその場で治り職場に復帰した。

コメント:
美点発見は、会社組織はもちろん、学校や教育機関でも多大な効果を発揮すると思います。
なぜなら、身の回りにいる家族や仲間たちとの人間関係が自然と良くなっていくからです。
今回の美点発見プロジェクトは、これからの学校教育を大きく変革するきっかけになると思いますので、
ご支援の程、よろしくお願いいたします。
```
