## 🎬 撮影・制作のヒント

### 実写 vs アニメーション

#### 実写の場合
```
メリット:
✅ リアリティ
✅ 共感しやすい
✅ 表情で感情が伝わる

デメリット:
❌ コストがかかる
❌ モデルが必要
```

#### アニメーション/モーショングラフィックスの場合
```
メリット:
✅ 制作しやすい
✅ イメージを自由に表現
✅ 修正が容易

デメリット:
❌ 温かみが出しにくい（工夫次第）
```

**推奨:** モーショングラフィックス + 実写の組み合わせ
- テキスト部分: モーショングラフィックス
- アプリ画面: 実写（実際のアプリ）
- 背景: 美しい実写映像
